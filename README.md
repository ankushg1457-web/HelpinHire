# HelpinHire

A modern, responsive recruitment website built with React and Tailwind CSS.

## 🎨 Brand Assets

HelpinHire now includes a complete logo package! Both assets are production-ready and available for download:

- **logo.svg** - Full brand logo with circular badge design
- **logo-icon.svg** - Icon-only version perfect for favicons and social media

📁 **Location**: `/public/` directory  
📖 **Details**: See [BRAND_ASSETS.md](./public/BRAND_ASSETS.md) for complete usage guidelines and specifications

### Quick Usage
```html
<!-- Full Logo -->
<img src="/logo.svg" alt="HelpinHire Logo" />

<!-- Icon (Favicon) -->
<link rel="icon" type="image/svg+xml" href="/logo-icon.svg" />
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
